# git使用手册

  • 查看状态git status
  • 提交前git add . 或者 git add a.txt
  • 提交的备注信息git commit -m '这里是备注信息'
  • 提交git push -u origin master
  • 拉取远程代码git pull origin master
  • 查看分支git branch -a
  • 新建分支git branch 分支名
  • 删除分支git branch -d 分支名
  • 切换分支git checkout 分支名
  • 合并分支,把某个分支合并到master主分支上git merge 分支名
  • 克隆远程仓库git clone 远程仓库的地址
  • 添加远程仓库地址git remote add origin 地址
  • 查看日志git log
